Introduction

This portfolio project aims to analyze the relationship between students' performance on tests and various demographic and socio-economic factors. We will explore the effects of gender, ethnicity, parental education, lunch quality, and test preparation courses on student scores.

Project Overview

We will use a dataset from Kaggle, preprocess and clean the data, and conduct exploratory data analysis (EDA) to gain insights. We will then develop a regression model using algorithms such as Decision Trees, Random Forest, Support Vector Regression (SVR), XGBoost, and CatBoost. The model will be hyperparameter-tuned and the best model selected. The final model will be deployed using Flask, and the code will be written as a modular, production-grade, deployable code with CI/CD MLOps integration from VSCode, GitHub Actions, Jenkins, and AWS.

Implementation Detail

  • Data Collection: Collect dataset from Kaggle
  • Data Preprocessing and Cleaning: Clean and prepare dataset for analysis
  • Exploratory Data Analysis (EDA): Gain insights into relationships between factors and student performance
  • Model Development: Develop regression model using algorithms such as Decision Trees, Random Forest, SVR, XGBoost, and CatBoost
  • Hyperparameter Tuning: Tune the hyperparameters of our models to improve their performance
  • Best Model Selection: Compare the performance of each model and select the best one to use in our project
  • Model Deployment using Flask: Deploy our model using Flask, a Python web framework. This will allow us to create a web application where users can input student data, and our model will predict their scores
  • Production Grade Code: Write our code as a modular, production-grade, deployable code with CI/CD MLOps integration from VSCode, GitHub Actions, Jenkins, and AWS
